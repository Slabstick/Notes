## Starten

- Ohne "scripts" in package.json:

```zsh
npx next dev
```

- Mit "dev" "scripts" in package.json:

```zsh
npx run dev
```

``` json
{
	"scripts": {
		"dev": "next dev"
	}
}
```

- Dev & Start:

```json
{
	"scripts": {
			"dev": "next dev",
			"build": "next build",
			"start": "next start"
		}
}
```

## Installieren

### Create-next-app:

```zsh
npx create-next-app@latest
```

### Manuell: 

- package.json erstellen:

```json
{
	"name": "next-reviews",
	"private": true,
	"scripts": {
		"dev": "next dev"
	}
}

```

```zsh
npm install next react react-dom
```

### Layout.jsx

- /app/layout.jsx als root layout datei:

```jsx
export default function RootLayout({ children }) {
	return(
		<html lang="en">
			<body>
				<header>
					[header]
				</header>
				<main>
					{children}
				</main>
				<footer>
					[footer]
				</footer>
			</body>
		</html>
	);
}
```

### Page.jsx

- /app/page.jsx als root homepage:

```jsx
export default function HomePage() {
	return (
		<h1>My first Next.js Page!</h1>
	)
}
```

## Routing

- Im **app Folder** bekommt jede Seite ihren **eigenen Folder** und eine "**page.jsx**" Datei.
	- Unterseiten bekommen ebenfalls eigene Unterordner mit eigener **page.jsx** Datei.
